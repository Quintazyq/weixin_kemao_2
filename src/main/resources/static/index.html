这个是静态的页面。如果要更新页面则可以直接通过AJAX访问RESTFUL风格的WEB服务器获取JSON数据，根据数据来更新页面。
static目录是特殊的目录，它表示静态资源，html、css、js等文件都可以放在此目录中。
在spring boot里面除了static目录能够这样做，在相同位置还可以创建public、resources目录、作用相同。
src/main/java 用于存储非.java文件
src/main/resources/static 用于存储静态的页面资源文件，html、css、js等。
src/main/resources/templates用于存储Thymeleaf模板文件。
src/test/java用于存储单元测试相关的.java文件。
src/test/resources用于存储单位测试相关的非.java文件。
